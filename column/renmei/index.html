【図解】夫婦連名・3名以上の連名でのご祝儀袋の書き方

更新日: 2026.05.04

職場の同僚と、あるいは夫婦揃って結婚式に参列する際、ご祝儀袋を「連名」で準備することがありますよね。
いざ筆を持った時、「夫と妻、どっちが右だったっけ？」「中央寄せと左右均等、どっちが正解？」と、ふとペンが止まってしまうことはありませんか？

マナーは「〇〇の法則」のような名前のついた堅苦しい定理ではなく、人と人との繋がり（ご縁）を大切にするための形です。受け取る新郎新婦が後から整理しやすいように、という思いやりの心遣いでもあります。
この記事では、迷いがちな連名の書き方を、人数別の図解で分かりやすく解説します。

📖 この記事の目次
  1. 夫婦連名で贈る場合（2つの書き方）
  2. 3名までの連名で贈る場合（友人・同僚）
  3. 4名以上の連名で贈る場合
  4. 会社・部署からの連名で贈る場合

1. 夫婦連名で贈る場合

夫婦で結婚式に招待された場合、一つのご祝儀袋に二人分の名前を連名で記載します。

A. 中央寄せ型

夫のフルネームを中央に書き、左隣に妻の名前のみを添える。

B. 左右均等型

中央線を挟んで両名のフルネームをバランスよく並べる。

2. 3名までの連名で贈る場合

格上の人（年長・役職上）を一番右に書き、順に右から左へ並べます。

3. 4名以上の連名で贈る場合

代表者の名前を中央に書き、その左下に「外一同」と添えます。全員の名簿は別紙に書き中袋へ同封しましょう。

4. 会社・部署からの連名で贈る場合

「○○部一同」と中央に記載するのが基本です。

書き方がわかったら、お手本を作ってみましょう！
